1. 封面：Agentic OCR 实战，营业执照从图片到可信企业信息
2. 目录：业务背景、Agent OCR 方案、Demo 演示、沉淀 Skill

## Demo 讲述句

这张营业执照不是只做 OCR，而是让 Agent 完成一个业务任务：提取企业信息，并判断哪些字段可信、哪些字段需要修复或人工复核。
